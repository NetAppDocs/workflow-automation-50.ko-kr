    мғҲ мӮ¬м „ н•ӯлӘ© лҢҖнҷ” мғҒмһҗм—җм„ң мғҲ мӮ¬м „ к°ңмІҙлҘј л§Ңл“Өкі  к°ңмІҙмқҳ м •мқҳлҘј м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • * к°ңмІҙ мң нҳ• мқҙлҰ„ *

      мӮ¬м „ к°ңмІҙмқҳ мқҙлҰ„мқ„ м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • * м„ӨлӘ… *

      мӮ¬м „ к°ңмІҙм—җ лҢҖн•ң м„ӨлӘ…мқ„ м ңкіөн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • * кө¬м„ұн‘ң *

      мҳӨлёҢм қнҠёмҷҖ м—°кҙҖлҗҳм–ҙм•ј н•ҳлҠ” мҠӨнӮӨл§ҲлҘј м„ нғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқёмҰқлҗң кө¬м„ұн‘ңм—җ мӮ¬мҡ©мһҗ м§Җм • мӮ¬м „ н•ӯлӘ©мқ„ м¶”к°Җн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • * м—”н„°нӢ° лІ„м „ *

    мҶҚм„ұ н…Ңмқҙлё”

    мӮ¬м „ к°ңмІҙлҘј м •мқҳн•ҳлҠ” нҠ№м„ұ лӘ©лЎқмқ„ м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • * мқҙлҰ„ *

      мҶҚм„ұ мқҙлҰ„мқ„ мһ…л Ҙн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • * мң нҳ• *

      нҳ•мӢқ кІҖмӮ¬лҘј мң„н•ҙ л¬ёмһҗм—ҙ, л¶Җмҡё л°Ҹ м •мҲҳ к°ҷмқҖ нҠ№м„ұмқҳ нҳ•мӢқмқ„ м„ нғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. м„ нғқн•ң кө¬м„ұн‘ңмқҳ мӮ¬м „ н•ӯлӘ©мқ„ м°ёмЎ°н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

    • * л¬ёмһҗм—ҙ кёёмқҙ *

      мҶҚм„ұм—җ лҢҖн•ҙ л¬ёмһҗм—ҙ мң нҳ•мқ„ м„ нғқн•ң кІҪмҡ° л¬ёмһҗм—ҙмқҳ кёёмқҙлҘј мҲ«мһҗлЎң м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • * к°’ *

      мҶҚм„ұм—җ лҢҖн•ҙ enum нҳ•мӢқмқ„ м„ нғқн•ң кІҪмҡ° мүјн‘ңлЎң кө¬л¶„лҗң к°’ лӘ©лЎқмқ„ м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • * лӮҙмё„лҹҙ нӮӨ *

      мӮ¬м „ нҠ№м„ұмқҙ мӮ¬м „ к°ңмІҙмқҳ кё°ліё нӮӨм—җ мҶҚн•ҳлҠ”м§Җ м—¬л¶ҖлҘј м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

      нҠ№м„ұмқ„ мәҗмӢңн• м§Җ м—¬л¶ҖлҘј м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
