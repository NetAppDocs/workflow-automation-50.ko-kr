= 타사 제품 업그레이드
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Windows의 JRE, MySQL 및 ActiveState Perl과 같은 타사 제품은 WFA(OnCommand Workflow Automation)에서 업그레이드할 수 있습니다. JRE, MySQL 등과 같은 타사 제품에서 보안 취약점이 보고됩니다. 이번 WFA 릴리즈부터 이제 자체적으로 타사 제품을 업그레이드할 수 있습니다.
